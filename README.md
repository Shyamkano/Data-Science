## 📌 Overview
This repository contains a comprehensive collection of hands-on Data Science exercises. It covers the end-to-end data pipeline: from **loading and cleaning data** to performing rigorous **statistical hypothesis testing** and **feature engineering**.

Key highlights include analyzing real-world datasets like **Stranger Things TV Show dialogues**, **World Bank GDP data**, and **Indian Cuisine ingredients**.

## 📂 Repository Structure

| Directory | Description | Key Concepts |
| :--- | :--- | :--- |
| **`/ANOVA`** | Analysis of Variance studies | One-way & Two-way ANOVA, F-statistic |
| **`/Hypothesis Testing`** | Statistical inference on datasets | T-tests (Paired/Ind.), Chi-Square, P-value interpretation |
| **`/Feature Scaling...`** | Preparing data for ML models | Standardization, Normalization, One-Hot Encoding, Imputation |
| **`/Data Frames...`** | Data Ingestion & Wrangling | Reading JSON/SQL/Excel, Data generation, Basic cleaning |
| **`/Datasets`** | Raw data files | CSV, JSON, and Excel datasets used in the notebooks |

## 🛠️ Tech Stack
- **Core:** Python 3.x
- **Data Manipulation:** Pandas, NumPy
- **Statistics:** SciPy, Statsmodels
- **Machine Learning:** Scikit-Learn (Preprocessing, Imputation, K-Means)
- **Visualization:** Matplotlib, Seaborn
