# 🐼 Data Frames & Basic Pre-processing

The foundation of any data science project is loading data correctly. This directory demonstrates how to ingest data from various file formats and sources.

## 📓 Notebooks

### `Data Frames and Basic Pre-processing.ipynb`
A handbook for Data Ingestion using Pandas.

**Data Sources Covered:**
- **CSV:** `dataset.csv`
- **Excel:** `Supermarket-Data.xlsx` (using `openpyxl`)
- **JSON:** `iris.json`
- **SQL:** Reading from `Car_Database.db` using `sqlalchemy`.
- **Scikit-Learn Datasets:** Generating synthetic regression/classification data and loading the `digits` dataset.

**Key Operations:**
- DataFrame inspection (`head`, `shape`).
- Handling different file encodings and orientations.
