# 💾 Datasets

This directory contains the raw data used across all practicals in this repository.

| Filename | Description | Used In |
| :--- | :--- | :--- |
| `stranger_things_all_dialogue.csv` | Dialogue scripts, timestamps, and stage directions. | Hypothesis Testing, ANOVA |
| `world_bank_dataset.csv` | GDP, Population, CO2 emissions, and Life Expectancy. | Feature Scaling (Numerical) |
| `cardata.csv` | Car sales data (Price, Year, Fuel Type). | Feature Scaling (Categorical) |
| `indian_food.csv` | Ingredients, prep time, and region of Indian dishes. | Exploratory Analysis |
| `dataset.csv` | Student mental health survey data. | Basic Pre-processing |
| `iris.json` | Standard Iris flower dataset in JSON format. | Basic Pre-processing |
| `Supermarket-Data.xlsx` | Sales data in Excel format. | Basic Pre-processing |
| `lung_cancer_survey.csv` | Health metrics and survey responses. | Numerical Preprocessing |
